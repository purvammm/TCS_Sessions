Spring Boot Training Series

Interactive slide decks and hands-on lab sheets — from a first Java program to a running microservices architecture. Authored by Purvam Prajapati, Faculty — AHD.

How to use Each session has a Theory file (an interactive slideshow — use the on-screen arrows or your keyboard's ← / → keys) and a Lab file (a scrollable companion with a progress checklist and copy-to-clipboard code). Every file is self-contained — just open it in any browser, no build step needed. If you're viewing this on GitHub, open index.html via a preview such as htmlpreview.github.io or GitHub Pages to get the clickable links below.
